# Network Troubleshooting Tool

A Python-based console application designed to assist IT support engineers in diagnosing basic network issues such as connectivity problems and DNS resolution errors.

## Features
- Check internet connectivity
- Diagnose DNS resolution issues
- Simulate basic network troubleshooting steps
- User-friendly console interface

## Technologies Used
- Python 3
- Socket module
- OS module

## How to Run
1. Clone the repository
2. Ensure Python 3 is installed
3. Run the program using:
   ```bash
   python network_tool.py
